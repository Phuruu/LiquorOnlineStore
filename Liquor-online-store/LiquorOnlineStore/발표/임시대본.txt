*** 메인 상품검색기능 - 메인 소개할 때 할지 상품 list 설명할 때 할지 정하고 넣기! ***

1. 메인 소개
[메인]
관리자를 제외한 모든 페이지에 공통적으로 들어가는 헤더부분과, 메인컨텐츠 부분, 풋터 만들었습니다. 
헤더의 경우 크게 유저, 메인, 카테고리 세 분류로 나누어집니다. 

헤더의
  유저 부분은 로그인과 회원가입, 마이페이지, 장바구니 로 구성되어있습니다. 로그인 하면 닉네임으로 바뀜과 동시에 제일 끝에 로그아웃 버튼도 생성됩니다. 
  메인 부분은 희로애락의 로고와 검색 바로 이루어져 있습니다. 로고 클릭 시 메인 화면으로, 검색 바에 상품 입력 시 검색 페이지로 이동합니다.
  카테고리 부분은 총 7개로 나누어져 있고, 전체 상품부터 QnA까지 다양하게 구성되어 있습니다. 

바로 아래의 메인컨텐츠 부분은
  이벤트 배너가 움직이고, 카테고리 헤더에 없는 상품의 상세 카테고리들이 나열되어 있습니다. 저희 희로애락은 탁주, 약청주, 과실주, 증류주. 총 네 분류로 전통주를 구분하고 있습니다. 해당 카테고리 클릭 시 그 분류의 상품들만 있는 페이지로 이동하게 됩니다. 
(# 스크롤 내리며)
  다음은, 희로애락의 전체 상품을 랜덤하게 보이며 상품을 추천합니다. 클릭하면 바로 해당 상품의 상세페이지로 이동합니다.
(#상세페이지 이동 후 메인 진입)

마지막으로 풋터는 법적인 정보 외 회사 관련 정보를 담고 있습니다. 그 중 이용약관, 개인정보처리방침을 누르면 관련 정보가 팝업창으로 뜨고,
(#각 버튼 클릭하여 팝업창 출력 확인)
(#입점신청 버튼 선택 후 구글폼 스크롤하여 보여주기)
 입점신청을 누르면 구글폼으로 이동하여 희로애락에 입점하고자 하는 업체의 신청서를 받습니다. 이렇게 들어온 신청은 이후 관리자가 확인하여 처리합니다.

2. 회원가입, 아이디/비밀번호 찾기
[회원가입]
헤더의 유저부분에 로그인/회원가입 버튼을 선택하면 (#버튼 선택) 로그인 창과 함께 아이디/비밀번호 찾기와 회원가입이 출력됩니다.
가장 먼저 회원가입을 진행(#회원가입 선택) 하면 회원의 정보를 입력할 수 있는 가입 폼이 나옵니다.

/* 비밀번호까지 입력 시 회원가입 버튼 누르면 alert 안내문구 출력됨 .. 어떻게 할 것인가
  유저가 회원가입할 때 하나라도 빈칸이 있으면 가입을 할 수 없으므로 하나라도 공백이 있을 시 가입되지 않게끔 설정해두었습니다. (#버튼 선택 시 회원가입 되지 않는 것 확인) 아이디, 닉네임, 비밀번호, 이메일, 핸드폰 번호에 각각 패턴을 주어 정규식에 부합하지 않을 경우 alert창으로 문구를 띄었습니다. 자세한 설명은 해당 부분에서 설명드리겠습니다.
*/

  아이디는 중복 체크를 통해 중복일 때 입력칸 하단에 이미 존재하는 아이디란 문구가 뜨고, (#test입력 후 중복확인)
중복이 아닐 경우 사용 가능한 아이디라는 문구가 뜨도록 설정했습니다. (#가입가능한 아이디 입력)
추가로 빈칸인 상태에서 중복체크 버튼을 누를 시 아이디를 입력하라는 문구가 뜹니다.
  아이디는 특수문자 조합이 불가능하며 입력 시 입력 불가능 안내문구가 출력됩니다.(#특수문자 넣어서 기능 보여주기-한글자 입력 시 alert 출력하니까 한개만 작성!)
  비밀번호는 영문, 숫자, 특수문자를 포함해야 하고, 최소 8글자에서 최대 16글자를 입력합니다.
비밀번호 실시간 체크를 통해 비밀번호와 비밀번호 확인란이 불일치할 경우 빨간색 배경을 띄워주도록 했는데 동시에 회원가입 버튼이 비활성화 되어 있는 상태를 확인할 수 있으며, (# 가입 비밀번호 입력 후 비활성회된 버튼 보여주기) 비밀번호와 비밀번호 확인란이 일치하게 되면 초록색 배경으로 바뀌게 됩니다.

//만약 빈칸 존재 시 안내문구 출력 보여주려면 여기다가 집어넣기!

닉네임은 아이디와 동일하게 중복체크 기능을 두었습니다. (#닉네임 입력 후 중복체크 선택)

  생년월일의 경우 주류 사이트이기 때문에 미성년자가 가입하지 못하도록 2004년까지만 선택 가능하도록 설정했습니다.
(#임의 생년월일 입력)
추가로 성별선택이 가능하며 (#성별 선택, 그대로 둬도 됨!)
핸드폰 번호는 ‘-‘(하이픈)을 제외한 최대 11글자까지 입력할 수 있도록 했습니다. 
주소는 주소 API를 가져와 우편번호를 찾을 수 있도록 설정해두었습니다. 

입력이 완료된 후 회원가입 버튼 선택 시 회원가입 동작이 완료됩니다.

[아이디 찾기/비밀번호 찾기]
아이디 찾기는 회원가입시 가입했던 이름과 핸드폰 번호를 입력해야 id를 찾을 수 있도록 설정해두었고, 만약 일치하지 않는 정보를 입력 시 ‘일치하는 정보가 존재하지 않는다’는 문구를 보여줍니다. (#정보 일치하지 않는 경우를 먼저 보여준 다음, 일치하는 정보 보여주기)
 
비밀번호 찾기의 경우 회원가입했던 아이디와 이메일을 받으며 일치하지 않는 정보를 입력할 시 ‘일치하는 정보가 없다’는 문구를 띄워줍니다. (#정보 일치하지 않는 경우를 먼저 보여준 다음, 일치하는 정보 보여주기)
(#일치하지 않는 정보 HONG, hong@TEST.COM / 일치하는 정보 HONG, HONG@TEST.COM)
일치하는 정보 입력 시 해당 회원의 비밀번호 출력 후 로그인 페이지로 진입합니다.

3. 일반 회원 로그인
[로그인]
기존에 가입되어 있던 회원의 정보로 로그인 하도록 하겠습니다.

4. 각 카테고리 소개 후 진입 확인(전체 상품 -> 종류별)
[전체 상품]
헤더 부분의 전체상품을 클릭하면 (#전체 상품 클릭)
저희가 판매하는 모든 상품의 리스트를 확인할 수 있고 여기서 상품하나를 클릭하면 클릭한 상품의 상세페이지를 확인할 수 있습니다.

[종류별]
 다시 메인으로 돌아와서, 메인슬라이드 아래를 보시면 탁주, 약.청주, 과실주, 증류주  4가지의 이미지 버튼이 있는데 해당 버튼을 클릭하시면 각자 카테고리에 맞는 상품의 리스트를 출력하게 됩니다.

(#탁주 클릭)
이렇게 탁주 버튼을 클릭하면 탁주 카테고리에 해당하는 상품들을 볼 수 있는데요,
여기서 다른 카테고리의 상품들이 보고 싶으면 위에 있는 버튼으로 간단하게 이동이 가능합니다.
(#이동하는거 보여주기)

그리고 상단 가운데에 보이는 셀렉트 옵션으로 필터 효과를 줬는데요, 총 3가지의 필터효과가 있습니다.
(#최신순 선택)
우선 최신순을 선택하면 저희가 상품을 등록한 순서대로 리스트를 출력하게 되고, 
(#가격높은순 선택)
가격높은순을 선택하면 가격이 높은순대로, 
(#가격낮은순 선택)
또 가격낮은순으로 선택하면 가격이 저렴한대로 출력이 되는걸 볼 수 있습니다.

마찬가지로 여기서도 상품을 클릭하면 해당 상품의 상세페이지를 확인할 수 있습니다.

[지역별]
헤더의 카테고리 중 지역상품을 선택해서 들어가게되면, (#카테고리 지역상품 클릭)
지역별로 생산한 전통주를 지도 이미지로 확인 가능합니다.
각 지역 선택 시 우측에 애니메이션 효과로 해당되는 전통주의 list가 출력, 확인되며 이미지 클릭 시 다른 카테고리와 마찬가지로 상세 페이지에 진입 가능하도록 해두었습니다. (#임의 상품 이미지 클릭)

5. 상세 페이지 진입
[상세페이지]
각 카테고리 진입 후 상품의 상세페이지로 진입하게 되면 개별 상품의 디테일한 정보를 보여줍니다. 
수량 변경하면 가격도 함께 변경되는 모습을 보실 수 있습니다.(#상품 수량 변경)
  페이지 진입 시 상품 수량의 기본값은 1이며, 1일 경우 마이너스 버튼이 비활성화 되어 있는 것을 확인하실 수 있습니다. 그리고 바로 구매하기 버튼 클릭 시 결제페이지로 가고, (#결제 버튼 클릭 후 다시 상세페이지로) 
장바구니 버튼 클릭 시 장바구니 테이블에 인서트 되면서 장바구니 이동과 관련된 팝업이 출력되는데 확인 선택 시 (#확인)
장바구니로 이동되며, (#전체상품 가서 꼭 다른 상품 선택!!!) 취소버튼 선택 시 상세페이지에 유지됩니다.

페이지 하단에는 상품 상세설명과 리뷰 확인, 배송 및 환불 관련 안내도 추가로 확인이 가능합니다. 

6. 장바구니 list 확인
[장바구니]
장바구니에서는 상세페이지에서 담은 상품들을 회원이 관리하기 쉽게 상품별 list로 출력하여 보여주며 상품의 개수를 변경한 뒤 결제 가능토록 구현하였습니다.
먼저 개수 항목의 버튼을 눌러 1~10까지의 상품 개수를 정할 수 있습니다.(#상품 개수 변경)
또한, 선택삭제, 전체삭제, 상품삭제 기능을 구현해 상품 앞쪽의 체크박스를 선택한 후 (#전체선택 해제 후 임의 상품 체크)
선택 삭제 버튼 클릭 시 (#버튼 클릭) 선택한 상품만 삭제되며 list 우측 상품 삭제 버튼을 누르면 (#상품삭제 버튼 클릭)
해당되는 하나의 상품만 삭제되고, 상단의 전체 삭제를 선택 시 상품 전체가 삭제됩니다.(#결제해야하니까 전체삭제 하지말까요?)

7. 장바구니 결제
[결제]
회원이 구매할 상품과 하단의 (#스크롤 내리며) 결제할 상품 개수, 총 주문금액을 확인 후 주문하기 버튼 선택 시 주문 페이지로 이동합니다. (#주문하기 버튼 선택)
  주문/결제 페이지는 상세페이지및 장바구니에서 넘어온 데이터를 바탕으로 총 결제금액과 각 물건들의 정보를 보여주며, 결제 시 사용할 쿠폰/할인을 설정하고 총 주문금액을 확인할 수 있습니다. 결제수단 선택 후 주문자 동의를 해야지만 해당 수단으로 결제가 가능합니다. (#체크박스 미선택 시 alert창 출력 확인)
체크박스 모두 선택 후 결제하기 버튼을 선택하면 각 결제수단 별로 API가 출력되는데, 무통장입금을 선택한 경우 kg이니시스를 통해(#무통장 입금 선택 후 결제) 결제가 가능하며,(#결제 중단 후 카카오페이 선택)
카카오페이를 선택한 경우 카카오페이 QR창이 (#카카오페이 선택 후 결제하기) 출력되며 결제 완료 시 메인페이지로 이동하게됩니다.
(#QR인식 후 메인페이지 이동)

주문은 카카오페이 다 보여주는지 카카오 결제 QR만 확인하고 돌아가는지..?

8. 커뮤니티 진입 후 설명
[커뮤니티]
메인 카테고리의 커뮤니티는 (#커뮤니티 진입)
회원들이 전통주에 대한 이야기를 자유롭게 나눌 수 있는 곳입니다.
커뮤니티는 게시판 형태로 만들었으며 한 페이지 당 10개의 글이 보이고 이를 넘어가면 페이징 처리를 하였습니다.
(#페이지 이동)
  가장 먼저 '글 등록'을 해보겠습니다. (#글 등록 클릭)
게시판은 희로애락의 회원들만 글을 작성할 수 있으며 작성자는 가입할 때 입력했던 닉네임이 표시되고 제목과 내용 그리고 수정, 삭제 시 필요한 비밀번호를 입력해야 작성이 가능합니다. 
  비밀번호는 숫자만 입력 가능하며 4글자 이상은 숫자가 적히지 않도록 구현했습니다. (#비밀번호 4글자 입력)
모든 값 입력 후 글 작성 버튼을 클릭(#글 작성버튼 클릭) 시 list에 방금 입력한 커뮤니티글이 보이는 것을 확인할 수 있습니다.
  작성된 글을 들어가면 제목, 작성자, 날짜, 내용이 보입니다.
회원들은 게시글을 보면서 하트 버튼을 눌러 공감 표시를 할 수 있으며(#공감버튼 클릭) 누를때마다 바로 알 수 있도록 옆에 공감 수를 표시하였습니다.
  또한 한 게시글마다 댓글을 작성할 수 있습니다. 댓글 또한 닉네임이 자동으로 표시되며 수정과 삭제시 필요한 비밀번호를 작성합니다. (#댓글 한 개 작성)
  댓글의 수정과 삭제도 해보겠습니다. 작성 시 기입한 비밀번호를 입력하고 수정 버튼을 누르면 수정이 가능합니다. (#비밀번호 입력 후 수정)
 마찬가지로 삭제도 비밀번호를 받아 바로 삭제가 가능합니다. (#비밀번호 입력 후 댓글 삭제)
  이제 게시글 수정과 삭제를 해보겠습니다. 
수정/삭제하기 버튼을 누르면 비밀번호 입력페이지로 넘어가고 (#수정/삭제 버튼클릭 후 비밀번호 입력)
 비밀번호를 입력하면 게시글 수정이 가능합니다. 이때 작성자와 날짜, 좋아요는 변경하지 못하도록 해두었고 제목과 내용만 수정이 가능 합니다. 
제목과 내용을 바꾼 뒤 수정버튼을 클릭하여 (#내용 변경 후 수정버튼 클릭) 다시 같은 게시물을 보면 수정된 것을 확인해 볼 수 있습니다. 또한 수정하기 옆에 있는 삭제하기 버튼을 눌러 이곳에서 삭제도 가능합니다. (#상세 재진입하여 해당글 삭제 후 list 보여주기)

9. 이벤트 페이지 진입하여 각 항목별 공지사항
[이벤트]
다음은 무료배송, 상품 추천 등 희로애락만의 특별한 이벤트를 한눈에 볼 수 있는 페이지 입니다. 상품 추천 배너를 클릭시 전체 상품 페이지 또는 상품 list로 이동하고, (#전체 상품페이지로 이동하는 이벤트 배너 클릭)
  무료배송과 쿠폰 이벤트 배너를 클릭하면 공지사항 페이지로 이동하는데 이때 쿠폰 코드와 함께 더 자세한 안내를 확인할 수 있습니다. (#공지사항 진입 배너 클릭)


10. 공지사항 진입
[공지사항]
방금 이벤트 항목에서 진입했던 것과 같이 카테고리에서도 공지사항을 선택 시 관리자가 직접 등록한 공지들을 회원들이 확인할 수 있습니다.
회원들은 제목을 선택 시 공지사항의 상세 내용을 확인할 수 있고, (#제목 선택하여 상세내용 확인)
  글 작성과 수정은 관리자 페이지에서만 가능합니다.

11. qna 진입
[qna]
메인의 마지막 카테고리인 qna페이지는 (#qna 페이지 진입)
회원이 상품,배송,입고문의 등 관리자에게 할 수 있는 모든 문의사항을 응답받을 수 있도록 만든 게시판입니다.
list에서 관리자의 답변 유무를 확인할 수 있도록 출력했고, 문의 제목은 공통적으로 가져가기 때문에 작성자로만 글을 검색할 수 있도록 해두었습니다. (#검색바에 홍길동 검색 후 전체 list 돌아오기)
  qna 등록 시 (#문의 등록 선택) 제목을 문의할 항목으로 선택하여 통일성을 주었으며, 로그인 정보로 작성자를 자동으로 가져오게끔 했습니다.
  작성 시 비밀번호는 숫자만 4글자까지 입력 가능하도록 제한을 두었고, (#비밀번호 4자리 입력 후 작성버튼 선택)
작성을 선택하면 문의가 list에 정상적으로 등록되는 것을 확인할 수 있습니다.
  제목을 선택하여 상세페이지 진입하면 (#제목 클릭) 
등록 시 작성한 비밀번호를 입력해야 상세페이지로 진입 가능하며, 일치하지 않는 비밀번호 입력 시 alert창이 출력되며 다시 입력 가능하도록 했습니다. (#일치하지 않는 비밀번호 입력 후 일치하는 비밀번호 입력하여 진입)
일치하는 비밀번호 입력 시 상세페이지로 진입 가능하며, 관리자의 답변을 확인할 수 있고, 문의 수정,삭제,목록이동이 가능합니다.
먼저 수정 버튼 클릭 시 (#수정버튼 클릭 -> 내용 변경 후 버튼선택)
문의 내용 수정이 가능하며 수정 후 버튼 선택시 정상적으로 수정된 것을 확인할 수 있습니다.
수정 옆에 있는 삭제버튼을 클릭 시 (#삭제버튼 선택) 글이 바로 삭제됩니다.

12. 마이페이지 진입
[마이페이지]
그 다음으로 헤더의 유저부분에 있는 마이페이지를 설명드리겠습니다, (#로그아웃 후 마이페이지 선택)
우선 로그인을 하지 않은채로 마이페이지를 클릭하면 로그인 먼저 해달라는 안내창이 뜨게 됩니다.
로그인 후 다시 마이페이지를 클릭하면 (#로그인 후 마이페이지 선택)
이렇게 마이페이지로 접속되는걸 확인할 수 있습니다.

마이페이지는 주문목록, 커뮤니티내역, 리뷰관리, 문의내역, 회원정보수정, 회원탈퇴로 구성되어 있는데 순서대로 설명드리겠습니다.

우선 마이페이지의 기본 화면인 주문목록을 보시면, 주문한 날짜가 상단에 표시되고 그 아래에 주문한 상품의 이미지, 이름, 가격, 주문수량, 총가격이 표시됩니다.

####### 주문목록 > 리뷰작성, 커뮤니티내역 아직 덜합침

구매 목록 우측의 리뷰작성 버튼을 누르면 (#리뷰작성 클릭)
 새 창에서 리뷰 작성 또한 가능합니다. 작성 창 내에 닉네임이 자동으로 입력되며 리뷰의 제목과 내용 그리고 1부터 5까지의 평점을 남길 수 있습니다. 간단하게 리뷰를 남겨 보도록 하겠습니다. (#임의의 리뷰 내용 입력) 
  모두 입력 후 리뷰 작성 버튼을 누르면 창이 꺼지고 리뷰가 작성됩니다. 
  리뷰는 상품 주문과 결제가 완료된 유저만 가능하도록 하기 위해 주문목록에서 작성할 수 있도록 했으며, 작성된 리뷰의 경우 마이페이지의 리뷰관리 메뉴를 통해서 확인할 수 있습니다. 

다음으로 보여드릴 세가지 항목은 내가 쓴 글을 보여주고 수정과 삭제가 가능하도록 만든 페이지들인데, 커뮤니티내역부터 보여드리겠습니다.

커뮤니티내역 메뉴에 들어가시면 내가 작성한 커뮤니티 글 리스트가 나오고 오른쪽에 있는 수정, 삭제 버튼을 클릭하면 커뮤니티 글 내용을 수정할 수 있습니다. (#수정버튼 클릭)
  이렇게 수정버튼을 누르면 새로운 창이 하나 생성되고 여기에서 게시글의 제목과 내용을 수정할 수 있습니다. 수정할 내용을 모두 작성하고 아래에 있는 수정버튼을 클릭하면 수정이 완료됩니다.
(#수정 완료하고 다시 수정창 들어가서 수정된 내용 보여주기)

그리고 다음으로 삭제버튼을 누르면(#삭제버튼 클릭)
 정말 삭제하겠냐는 안내창이 나오고 확인 버튼을 누르면 게시글이 삭제됩니다.

그리고 다음 리뷰작성과 문의내역도 같은 내용입니다. 시연으로 보여드리겠습니다.
(리뷰, QNA 수정삭제 시연보여주기)

다음 회원정보수정 보여드리겠습니다.
여기서는 변동값이 아닌 아이디는 변경 불가능하게 고정이 되고 변동 가능한 닉네임, 이메일, 전화번호 등은 수정할 수 있게 구성했습니다. (#정보 변경값 입력하고 수정버튼 클릭)
이렇게 정보값을 모두 입력 후 수정버튼 클릭 시 회원 정보가 수정됨을 확인할 수 있습니다.

다음으로 회원탈퇴 보여드리겠습니다.
회원탈퇴는 비밀번호를 입력하고 비밀번호가 일치하면 탈퇴가 되는 방식으로 만들었는데요,
(#회원 비밀번호 입력)
여기 진짜 탈퇴하시겠습니까라는 안내창이 나오고 확인을 클릭하면 탈퇴가 완료됩니다.
(#탈퇴한 아이디로 로그인해서 보여주기)

—-----------------관리자페이지

그리고 매출정보 부분에서는 먼저 그래프를 통해 현재연도의 매출현황을 보여주고 옆에는 기간별로 검색을 할 시 매출현황을 보여줍니다
매장관리부분에서는 제품관리와 리뷰관리로 나뉘며

회원관리, q&a부분 및 공지사항 부분은 똑같이 리스트화 했으며 삭제 관리기능 및 검색기능을 추가 하였습니다


—-----------------검색
다음으로 설명드릴 부분은 상품검색인데요, 헤더부분을 보면 검색창을 보실 수 있는데 여기에 찾고싶은 상품을 검색하면 검색어에 해당되는 상품들의 리스트가 출력됩니다.

(#막걸리 검색) 
예를들어 이렇게 막걸리를 검색하면 막걸리 검색 결과입니다 라는 문구가 표시되고 그 아래에 막걸리 단어가 포함된 상품들의 리스트를 확인할 수 있습니다.

다른것도 검색해보면, 
(#소주 검색)

만약 아무것도 입력하지 않고 검색한다거나 또는 저희 쇼핑몰에서 판매하지 않는, 
상관없는 단어를 검색한다면 
(#아무거나 검색) 
검색된 내용이 존재하지 않는다는 문구가 표시됩니다.
